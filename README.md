# 🌍 Earthquake Analysis Dashboard (Q1 2026)

## 📌 Project Overview

This project analyzes global earthquake events recorded during Q1 2026 using Python, SQL, and Power BI.

The objective is to transform raw earthquake data into meaningful insights by performing data cleaning, exploratory data analysis, SQL-based querying, and dashboard visualization.

This project demonstrates end-to-end Data Analytics skills including:

✔ Data Cleaning  
✔ Data Analysis  
✔ SQL Querying  
✔ Dashboard Development  
✔ Data Visualization  
✔ Insight Generation

---

## 🎯 Project Objectives

- Analyze earthquake occurrences across the globe.
- Categorize earthquakes based on magnitude levels.
- Identify the strongest earthquake events.
- Study daily earthquake activity trends.
- Create an interactive Power BI dashboard for data exploration.

## 📊 Dashboard Features

### 🌎 Global Earthquake Location Map

Visualizes earthquake occurrences across different regions of the world.

### 📅 Daily Earthquake Trend

Tracks earthquake activity throughout Q1 2026.

### 🏆 Top 10 Strongest Earthquake Events

Displays the most significant earthquake events based on magnitude and significance scores.

### 📊 Magnitude Category Distribution

Analyzes earthquake counts across:

- Micro
- Minor
- Light
- Moderate
- Strong
- Major

### 🎛 Interactive Filters

Allows users to explore data based on magnitude categories.

---

## 📈 Key Insights

### 1. Majority of Earthquakes are Micro Magnitude

Most recorded earthquake events belong to the Micro category.

### 2. High Activity Around the Pacific Region

The earthquake map shows heavy seismic activity around the Pacific Ring of Fire.

### 3. Strongest Events

The strongest earthquake events were recorded near:

- South Sandwich Islands Region
- Kermadec Islands
- Fiji Islands
- Mid-Indian Ridge

### 4. Stable Daily Activity

Daily earthquake counts remained relatively consistent with occasional spikes.
